`cout << "Hello World!" << endl;`

이때 백틱기호 오른쪽에 해당 언어를 써주면 언어에 맞는 소스 형태로 표시됨

```C++
#include <iostream>
using namespace std;

int main(){
  cout << "Hello World!" << endl;
}
```
